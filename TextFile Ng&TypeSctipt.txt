//ActivatedRoute is used to get information about current route such as:
route parameter(id), query parameter, fragment, URL info

-It helps the component to know which data to show.

there 2 ways : 
1)using snapshot: Snapshot gives the current state of URL at the time component loads.
If value changes after component loaded, snapshot won’t update.  
ex: const id = this.route.snapshot.paramMap.get('id');

2)using observables:Route parameters are Observables because they can change while component is active (user clicks inside app).
So we subscribe to get values live whenever route changes.
ex:
this.route.params.subscribe((params) => {
  console.log(params['id']);
});

//see diffrence between these both in project : FirstDemo 
| Concept   | Shop Example                                                                                         |
| --------- | ---------------------------------------------------------------------------------------------------- |
| Snapshot  | You enter shop & see **price of shoes** once → If shopkeeper updates price later → You will not know |
| Subscribe | Shopkeeper updates price & **keeps informing** continuously                                          |



// Simple routing vs Lazy Loading Routing:

Simple routing ex: 
{
	path: 'Home',
    component: HomeComponent,
},

lazyLoading ex: 
{
    path: 'Home',
    loadComponent: () =>
      import('./home/home.component').then((c) => c.HomeComponent),
},

--> simple routing ma bdha component ek sathe j load thy jai jyare lazy loading ma only current route no ek j component load thse 
	aathi realoding time ghte ane app fast bne. in short jyare jeni jroor pde tyare j tene load kro.
